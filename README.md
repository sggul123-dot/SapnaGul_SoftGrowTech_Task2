# SapnaGul_SoftGrowTech_Task2
Stock Portfolio Tracker – Task 2
Description

This project is a simple Stock Portfolio Tracker developed in Python.
It allows the user to enter stock names and quantities, calculates the total investment value based on predefined stock prices, and optionally saves the result in a text file.

Features

User inputs stock name and quantity

Uses a hardcoded dictionary for stock prices

Calculates total investment value

Displays total result in console

Saves total investment value in a .txt file

Stock Prices Used

The following predefined stock prices are used:

AAPL: 180

TSLA: 250

GOOG: 2700

MSFT: 300

Concepts Used

Dictionary

User input/output

Basic arithmetic operations

File handling

Object-Oriented Programming (Class)

How to Run

Open terminal or command prompt.

Navigate to the project folder.

Run the command:

python StockPortfolioTracker.py

Enter stock name and quantity when prompted.

Type "done" to finish and see total investment value.
